Simple Python Flask REST API

Pre-requisites:
		Python 3
		Flask
		JWT
		SQLAlchemy
		Flask-Restful

How to use?
1. Run using python app.py

2. Has following endpoints

		GET - /items
		GET - /items/<name>
		POST - /items/<name>
		PUT - /items/<name>
		DELETE - /items/<name>
		POST - /auth
		POST - /register
		GET - /stores
		GET - /store/<name>
		POST - /store/<name>
		DELETE - /store/<name>
