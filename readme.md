# The project is Python Flask REST API for a store.

## Pre-requisites and Installation:
		Python 3
		Flask
		JWT
		SQLAlchemy
		Flask-Restful
		'''
		pip install Flask
		'''

# How to use?
		Run using "python app.py"

## Has following endpoints

		GET - /items
		GET - /items/<name>
		POST - /items/<name>
		PUT - /items/<name>
		DELETE - /items/<name>
		POST - /auth
		POST - /register
		GET - /stores
		GET - /store/<name>
		POST - /store/<name>
		DELETE - /store/<name>
