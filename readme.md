# The project is Python Flask REST API for a Store.

## Pre-requisites:
	Python 3
	Flask
	JWT
	SQLAlchemy
	Flask-Restful

## Installation
	pip install Flask

### How to use?
	Run using "python app.py"

### Has following endpoints

	GET - /items
	GET - /items/<name>
	POST - /items/<name>
	PUT - /items/<name>
	DELETE - /items/<name>
	POST - /auth
	POST - /register
	GET - /stores
	GET - /store/<name>
	POST - /store/<name>
	DELETE - /store/<name>
