## The project is Python Flask REST API for a Store

### Pre-requisites:
	Python 3
	Flask
	JWT
	SQLAlchemy
	Flask-Restful

### Installation
	pip install Flask

### How to use?
	Run using "python app.py"

### Has following endpoints
	GET - /items
	GET - /item/<name>
	POST - /item/<name>
	PUT - /item/<name>
	DELETE - /item/<name>
	POST - /auth
	POST - /register
	GET - /stores
	GET - /store/<name>
	POST - /store/<name>
	DELETE - /store/<name>
